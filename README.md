# Nuestro Pueblo

Sitio web cultural, turístico y de entretenimiento dedicado a contar historias,
tradiciones y postales de Jesús María y Colonia Caroya (Córdoba, Argentina).

El proyecto reúne relatos, imágenes y eventos locales. El contenido enviado por
usuarios se publica solo tras revisión del administrador.

## Secciones principales

- Inicio: presentación general y acceso a contenidos destacados.
- Lugares: reseñas de sitios históricos y recorridos culturales.
- Costumbres: fiestas y tradiciones de la región.
- Historia: museo, patrimonio e identidad local.
- Eventos: calendario cultural y festivalero.
- Personajes: fotografías y memorias de protagonistas locales.
- Contacto: envío de mensajes e imágenes para revisión.
- Privacidad: políticas, términos y condiciones del sitio.

## Características destacadas

- Diseño unificado con tipografía y colores del sitio.
- Vistas con imagen destacada y lightbox para ampliar fotografías.
- Formulario de contacto con validación y vista previa de imagen.
- Enlaces de navegación y rutas con React Router.

## Scripts

```bash
npm start
npm run build
npm test
npm run check:encoding
```

## Encoding (UTF-8)

Para evitar problemas de acentos y caracteres especiales, el proyecto fuerza
UTF-8 con `.editorconfig`. Además, existe un chequeo rápido para detectar
caracteres dañados:

```bash
npm run check:encoding
```

## Notas de contenido

El sitio tiene un fin cultural y familiar. El administrador se reserva el
derecho de publicar, editar o rechazar contenido enviado por usuarios.
